# Public Source Inputs Not Copied

Generated: 2026-05-16 18:52:09 AEST

The paper uses public raw/source data: FHWA 1995 Highway Finance tables, Census Journey-to-Work data, Census PUMS/PUMA data, American Travel Survey data used only in preliminary analysis, Census land/population tables, and Rae (1969) toll-road-mile historical data.

This package does **not** copy the large raw public Census PUMS `.Z` files or California PUMS StuffIt archives. It retains the authored/local derived data products, extraction scripts, lookup tables, small derived PUMS outputs, regression datasets, and workbooks needed to document how the published empirical model was assembled.

See `metadata/PUBLIC_RAW_INPUTS_NOT_COPIED.csv` for the local filenames, sizes, and hashes of raw public inputs that were intentionally excluded.
